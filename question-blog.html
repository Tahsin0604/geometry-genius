What is DOM and what is the purpose of DOM?

Full meaning of DOM is Document Object Model. It is a programming interface for web documents.
Programmer use DOM to use and manipulate webpage structure,style and content.

2.getElementsByClassName()

4.querySelector()
5.getElementById()

What is event bubble?

Event Buubble is a concept in the DOM. It executes when an element receives an event and that event propagates to parent element of DOM tree untill it reaches to root element.

What is a callback function and why will you use it?

Callback Function is a function where we passed a function as parameter which will excute after another function has finished execution.
Beacuse many JavaScript actions are asynchronous, which means they don't really stop the program from running until they're completed. thats why we need callback function to sequnece our output.
